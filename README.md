# Task List Utility

A simple Task List Utility built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

task-list-utility/
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
